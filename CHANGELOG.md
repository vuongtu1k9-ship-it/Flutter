# CHANGELOG

### Added
- Initial Flutter Xiangqi game implementation
- Game board with all pieces (General, Advisor, Elephant, Chariot, Cannon, Horse, Soldier)
- Basic game rules and piece movement
- Undo functionality
- Localization support (English and Vietnamese)
- Material 3 design with custom board painter

### Changed
- Updated pubspec.yaml to include flutter_localizations and intl packages
- Enhanced main.dart with localization support
- Improved game state management

### Fixed
- Initial board setup
- Game over detection
- Player switching logic
