# OpenClaw Workspace

## Overview
Multi-agent system with Telegram integration.

## Quick Start

1. Start gateway: `openclaw gateway start`
2. Check status: `openclaw status`
3. Send message via Telegram with @Linh020182_bot

## Files

- `AGENTS.md` - Agent configuration
- `SOUL.md` - Agent personality
- `BOOTSTRAP.md` - Initialization
- `OUTPUT.md` - Output contract
- `ACTION_PLAN.md` - Current goals
- `SETUP.md` - Setup guide
- `ARCHITECTURE.md` - System architecture

## Commands

```bash
openclaw gateway restart
openclaw status
openclaw logs --follow
```
